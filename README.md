# sketch-tools

> A collection of tools for Sketch

## Install

```sh
npm install sketch-tools
```

## Usage

```js
import { getDocument } from 'sketch-tools'

const document = getDocument()
```

## 目标
- [] 操作层
  - [] 选中层
  - [] 参考层

- [] 拖动块

- [] 缩放块
